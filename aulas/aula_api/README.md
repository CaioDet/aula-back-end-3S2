# 🚀 Minha Primeira API com Node.js e Express

Bem-vindo(a) ao repositório da minha primeira API! Este projeto foi construído como parte dos estudos de **Desenvolvimento de Sistemas** e demonstra como criar um servidor Back-end do zero usando JavaScript.



## 🎯 Sobre o Projeto
Este código é a estrutura básica de uma API RESTful. Pense nesta API como o "garçom" de um restaurante: ela recebe o pedido do cliente (Front-end), processa a informação e devolve a resposta no formato JSON.

## 🛠️ Tecnologias Utilizadas
* **Node.js**: O ambiente que nos permite rodar JavaScript no servidor.
* **Express**: O framework principal que organiza as rotas ("portas") da nossa API.
* **Body-Parser**: Um tradutor que ajuda a nossa API a entender os dados enviados pelo usuário.
