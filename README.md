# 🖥️ Desenvolvimento de Sistemas: Back-End

Bem-vindos ao repositório central das nossas aulas de **Back-End**! Este espaço foi criado para armazenar códigos-fonte, exemplos práticos e exercícios desenvolvidos em sala de aula.

O foco aqui é entender como a "mágica" acontece por trás das cortinas: servidores, bancos de dados, APIs e a lógica de negócios que sustenta as aplicações.

---

## 📚 Conteúdo Programático

Neste repositório, exploraremos os seguintes tópicos:

| Módulo | Descrição | Status |
| :--- | :--- | :--- |
| **01. Fundamentos** | Introdução ao Node.js, NPM e arquitetura Client-Server. | ✅ Concluído |
| **02. Versionamento** | Boas práticas de Git e GitHub para Back-End. | 🔄 Em andamento |
| **03. APIs REST** | Criação de rotas, métodos HTTP (GET, POST, PUT, DELETE). | 📅 Em breve |
| **04. Persistência** | Integração com bancos de dados (SQLite/SQL). | 📅 Em breve |

---

## 📂 Estrutura do Repositório

Para facilitar a localização dos arquivos, o repositório está organizado da seguinte forma:

* `/aulas`: Pastas numeradas com o conteúdo teórico e prático de cada encontro.
* `/exercicios`: Desafios semanais para fixação do conteúdo.
* `/projetos`: Modelos de projetos integradores.
* `/.gitignore`: Configurado para ignorar a pasta `node_modules` e manter o ambiente limpo.

---

## 🛠️ Tecnologias Utilizadas

* **Runtime:** Node.js
* **Gerenciador de Pacotes:** NPM
* **Banco de Dados:** SQLite
* **Versionamento:** Git
